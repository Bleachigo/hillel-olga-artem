## ДЗ 9.1 Розробка "Simple Site" за макетом Figma

Спільна робота Ольги та Артема.

### Використаний підхід

Був сворений "скелет" розмітки. Кожен з учасників зтянгнув собі на комп
актуальну версію з головної гілки, створив окрему гілку, працював на окремій гілці
та. Після виконання своєї частини гілки були злиті, результат перевірений. Після перевірки, було проведене злиття з головною гілкою та результат був наданий викладачу для перевірки.
